# PlantDiseaseDetection
Comparing performance of Shallow and Deep CNN models

---

Dataset: https://www.kaggle.com/datasets/abdallahalidev/plantvillage-dataset

From the above dataset, models are trained only on images related to Apple only, that is Healthy set: {'Apple___healthy'} & Diseased set: {'Apple___Apple_scab', 'Apple___Black_rot', 'Apple___Cedar_apple_rust'}
